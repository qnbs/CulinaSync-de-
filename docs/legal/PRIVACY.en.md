# Privacy Policy (CulinaSync)

> Updated: 2026-07-15 · Applies to the local-first PWA and optional Tauri desktop builds.

## Summary

CulinaSync stores household/recipe data **locally** on your device (IndexedDB / Dexie). There is **no** central CulinaSync account server. Cloud features (Gemini, WebDAV/Nextcloud, optional CDNs for on-device models) run only when you enable them or consciously send data there.

See the German canonical text in [`DATENSCHUTZ.md`](./DATENSCHUTZ.md) for the full legal wording (DE-first product).
